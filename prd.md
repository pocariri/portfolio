# 자기소개 홈페이지 PRD

## 1. 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 서승혁 자기소개 홈페이지 |
| 목적 | 자기소개와 포트폴리오를 통해 '서승혁'이라는 사람을 알리기 위한 개인 소개 페이지 |
| 사용 맥락 | 부트캠프에서의 간단한 자기소개 용도 |
| 작성일 | 2026-06-30 |

## 2. 타깃 사용자

- **부트캠프 강사님들**: 수강생의 기술 역량과 프로젝트 경험을 파악하려는 목적
- **부트캠프 동기님들**: 함께 배우는 동료로서 서로를 알아가려는 목적

> 두 사용자 모두 IT/개발에 친숙한 집단이므로, 기술 스택과 프로젝트 내용을 진솔하고 명확하게 전달하는 데 초점을 둔다. 과장된 마케팅 톤보다 "이 사람이 무엇을 만들 줄 알고 어떤 성향인지"가 잘 드러나야 한다.

## 3. 핵심 목표

1. 방문자가 페이지를 둘러본 뒤 **서승혁이 어떤 사람인지** 직관적으로 이해할 수 있다.
2. 포트폴리오를 통해 **프론트엔드·백엔드·모바일을 두루 다루는 역량**이 드러난다.
3. **차분하고 세련되면서도 화려한** 첫인상을 남긴다.

## 5. 디자인 컨셉

### 5.1 무드
- **미니멀과 화려함의 공존** — 깔끔한 레이아웃 위에 화려한 디테일(그라데이션, 잔상 효과)을 얹는다.
- 전반적으로 **밝고 차분하며 세련된** 분위기.

### 5.2 컬러
- 메인 컬러: **연보라 + 연분홍** (예: `#D8B4FE`, `#FBCFE8` 계열)
- 라이트모드를 기본으로 하되, **다크모드도 지원**.
- 배경은 정적이지 않고 **계속 움직이는 그라데이션** — **좌우(가로) 방향**으로 은은하게 흐르는 애니메이션.

### 5.3 타이포그래피
- **고딕 계열의 각진 글자체** 사용 (예: Pretendard, Noto Sans KR, Spoqa Han Sans 등 산세리프 고딕).
- 언어: **한글** (1차 기준).

## 6. 인터랙션 / 애니메이션

| 효과 | 설명 |
|------|------|
| 부드러운 스크롤 | 섹션 간 이동 및 휠 스크롤이 매끄럽게(smooth scroll) 동작 |
| 마우스 잔상 | 커서를 따라다니는 잔상(트레일) 효과 |
| 메뉴 호버 효과 | 메뉴에 마우스를 올리면 시각적 강조(언더라인·색상 변화 등) |
| 메뉴 호버 확대 | 메뉴에 마우스를 올리면 커서가 살짝 확대 |
| 움직이는 그라데이션 | 배경 그라데이션이 좌우로 끊임없이 흐르듯 이동 |

> 애니메이션은 "차분하지만 멈춰 있지 않은" 인상을 목표로 한다. 과하지 않게, 그러나 화면이 살아있다는 느낌을 주도록 조율한다.

## 7. 기술 스택

- **구현 방식**: **React 기반 SPA** (확정)
- **애니메이션**: CSS transition/animation + (필요 시) Framer Motion, GSAP
- **마우스 잔상·커서 효과**: JS 기반 커스텀 커서 구현
- **다크모드**: CSS 변수 + 토글 버튼 (라이트모드 기본)
- **폰트**: 고딕 계열 웹폰트(Pretendard 등) CDN 로드
- **이미지 경로**: 프로젝트 루트의 `images/` 폴더 참조
- **배포**: GitHub Pages / Vercel / Netlify 중 선택
